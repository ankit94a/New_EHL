About EHL

The battle worthiness of the Army is dependent on its equipment state. Hence, a need was felt to empower both the users and the maintenance agency to address equipment-related issues effectively and instantaneously.

Thus, there was a need for the creation of a digital knowledge database and repository which facilitates knowledge management for the Indian Army via a web-based application, hosted on Army intranet.
